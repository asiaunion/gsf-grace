### 작업 내용

#### 4-1. story/index.astro 폼 교체 (JA)

기존 `<form>` 블록을 아래로 교체:

```html
<div class="newsletter-wrap" id="newsletter-ja">
  <form class="inline-form" id="kit-form-ja" novalidate>
    <input
      type="email"
      name="email"
      placeholder="メールアドレス"
      required
      autocomplete="email"
      id="kit-email-ja"
    />
    <button type="submit" id="kit-btn-ja">手紙を受け取る</button>
  </form>
  <p class="form-msg" id="kit-msg-ja" aria-live="polite"></p>
</div>
```

`<script>` 추가 (페이지 하단):
```html
<script>
  const form = document.getElementById('kit-form-ja');
  const msg = document.getElementById('kit-msg-ja');

  form?.addEventListener('submit', async (e) => {
    e.preventDefault();
    const email = (document.getElementById('kit-email-ja') as HTMLInputElement).value.trim();
    if (!email) return;

    const btn = document.getElementById('kit-btn-ja') as HTMLButtonElement;
    btn.disabled = true;
    btn.textContent = '送信中…';

    try {
      const formId = import.meta.env.PUBLIC_KIT_FORM_ID;
      const apiKey = import.meta.env.PUBLIC_KIT_API_KEY;
      const res = await fetch(
        `https://api.convertkit.com/v3/forms/${formId}/subscribe`,
        {
          method: 'POST',
          headers: { 'Content-Type': 'application/json' },
          body: JSON.stringify({ api_key: apiKey, email }),
        }
      );
      if (res.ok) {
        msg.textContent = '登録しました。東京からの手紙をお楽しみに。';
        msg.style.color = 'var(--color-accent)';
        form.style.display = 'none';
      } else {
        throw new Error('failed');
      }
    } catch {
      msg.textContent = 'エラーが発生しました。もう一度お試しください。';
      msg.style.color = '#e53e3e';
      btn.disabled = false;
      btn.textContent = '手紙を受け取る';
    }
  });
</script>
```
