## STEP 3. 언어 스위처 추가 (Nav.astro)

### 위치
- 데스크톱: nav-menu **우측 끝** (메뉴 5개 뒤)
- 모바일(<768px): **햄버거 밖에 독립 배치** — 메뉴를 열지 않아도 항상 보이게

### 동작 (Astro i18n API 사용)
```astro
---
import { getRelativeLocaleUrl } from 'astro:i18n';
const currentLocale = Astro.currentLocale ?? 'ja';
// 현재 경로에서 locale 접두사만 제거한 "순수 경로" 추출
const path = Astro.url.pathname.replace(/^\/ko/, '') || '/';
const jaUrl = path;                              // JA = 무접두사
const koUrl = '/ko' + (path === '/' ? '' : path); // KO = /ko 접두사
---
<div class="lang-switch" role="group" aria-label="言語 / 언어">
  <a href={jaUrl} class:list={['lang-opt', { active: currentLocale === 'ja' }]} lang="ja">日本語</a>
  <span class="lang-sep">|</span>
  <a href={koUrl} class:list={['lang-opt', { active: currentLocale === 'ko' }]} lang="ko">한국어</a>
</div>
```

### 스타일
- 비활성: `var(--color-muted)` / 활성: `var(--color-accent)` 굵게
- 작게(0.8rem), 구분자 `|`는 line 색
- hover: accent

## STEP 5. hreflang + i18n SEO (BaseLayout.astro)

`<head>`에 추가:
```astro
---
const path = Astro.url.pathname.replace(/^\/ko/, '') || '/';
const jaHref = new URL(path, Astro.site).href;
const koHref = new URL('/ko' + (path === '/' ? '' : path), Astro.site).href;
---
<link rel="alternate" hreflang="ja" href={jaHref} />
<link rel="alternate" hreflang="ko" href={koHref} />
<link rel="alternate" hreflang="x-default" href={jaHref} />
```
- `<html lang={lang}>` 이미 적용됨 — 유지
- canonical은 현재 URL 유지 (이미 있음)
