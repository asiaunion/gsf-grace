## STEP 1. 메뉴 좌측 정렬 (Nav.astro)

### 원인
`.nav-brand`의 `margin-right: auto`가 로고와 메뉴 사이를 최대로 벌려 메뉴를 우측으로 밀어냄.

### 수정
```css
/* .nav-brand 에서 margin-right: auto 제거 */
.nav-brand {
  display: flex;
  flex-direction: column;
  line-height: 1.2;
  text-decoration: none;
  flex-shrink: 0;
  /* margin-right: auto;  ← 이 줄 삭제 */
}

/* .lang-switch 에 margin-left: auto 추가 (스위처만 우측 끝으로) */
.lang-switch {
  display: flex;
  align-items: center;
  gap: 0.3rem;
  flex-shrink: 0;
  order: 2;
  margin-left: auto;   /* ← 추가: 스위처만 우측 끝 */
}
```
- 결과 배치: **GSFGrace  메뉴5개 ………(여백)……… 日本語|한국어**
- 로고와 메뉴는 좌측에 자연스럽게 인접, 스위처만 우측 끝.
- `.nav-menu`는 현행 유지(`order:1; flex:0 1 auto`). 로고 옆에 gap(1.25rem)만큼 띄워 붙음.
- gap이 너무 붙으면 `.nav-menu`에 `margin-left: 1.5rem` 정도 추가 가능(선택).

> ※ 모바일(<768px) 현행 유지 — 건드리지 말 것.

## STEP 3. About placeholder 제거 (index.astro + ko/index.astro)

### 현재
About이 2단 그리드(좌: nihonbashi-placeholder.svg / 우: 텍스트).

### 수정: 이미지 제거, 텍스트 단독
```html
<!-- figure(about-photo) 블록 전체 삭제 -->
<section class="section-about">
  <div class="about-text">   <!-- about-grid 제거, about-text만 -->
    <p class="about-label">ABOUT GSFGRACE</p>
    <p class="about-body">東京・日本橋から、<br />恵みのうちに、ともに歩むために。</p>
    ...
  </div>
</section>
```
```css
/* about-grid의 2단 → 1단(또는 중앙 정렬) */
.section-about {
  padding: 3rem 0;
  max-width: 640px;   /* 텍스트 가독 폭 */
  margin: 0 auto;
}
/* .about-grid, .about-photo, .photo-img, .photo-caption 규칙 삭제 가능 */
```
- 정렬: 좌측 정렬 유지(현행) 권장. 또는 중앙 정렬 중 보기 좋은 쪽.
- 추후 실사진 준비 시 다시 2단으로 복원 예정 → 관련 CSS는 주석 처리해 보존해도 좋음.

> ※ Hero 수채화가 강렬하므로, 바로 아래 빈 회색 박스 제거로 완성도 상승.

## STEP 4. Hero 가독성 보강 (index.astro)

시안에서 부제·성구·보조버튼이 수채화 밝은 부분에 묻힘. 3가지 수정:

### 4-1. muted 텍스트 약간 진하게
```css
.hero-tagline { color: var(--color-text); }   /* 이미 text — 유지 */
.hero-eng     { color: #57534E; }   /* muted(#78716C) → 약간 진하게 */
.hero-verse   { color: #57534E; }   /* 동일 */
```

### 4-2. 오버레이 중앙값 상향 (텍스트 영역 보호)
```css
.hero::before {
  background: linear-gradient(
    to right,
    rgba(250, 248, 243, 0.93) 0%,
    rgba(250, 248, 243, 0.82) 50%,   /* 0.75 → 0.82 */
    rgba(250, 248, 243, 0.45) 100%   /* 0.28 → 0.45 */
  );
}
```

### 4-3. 보조 버튼(疑問に答える) 가시성 — 반투명 흰 배경
```css
.btn-secondary {
  background: rgba(255, 255, 255, 0.7);   /* transparent → 반투명 흰 */
  color: var(--color-text);                /* muted → text(진하게) */
  border: 1.5px solid var(--color-accent); /* line → accent(또렷) */
  backdrop-filter: blur(2px);
}
.btn-secondary:hover {
  background: rgba(255, 255, 255, 0.9);
  border-color: var(--color-accent-deep);
  color: var(--color-accent-deep);
}
```
