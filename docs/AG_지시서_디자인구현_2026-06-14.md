## 목표

현재 violet(#7c3aed) 기반 디자인을 **olive-earth 시스템**으로 전면 교체하고,
Manifesto 카피 세트를 적용한다.

**핵심 순서: ① 토큰 변수화 → ② 색 교체 → ③ 폰트 → ④ 카피 → ⑤ 사진 → ⑥ 검증**
순서를 지킬 것. ②를 ①보다 먼저 하면 색이 페이지마다 어긋난다.

## STEP 1. 전역 토큰 정의 (BaseLayout.astro)

`src/layouts/BaseLayout.astro`의 `:root`를 아래로 **완전 교체**:

```css
:root {
  --color-bg:          #FAF8F3;
  --color-surface:     #FFFFFF;
  --color-text:        #292524;
  --color-muted:       #78716C;
  --color-accent:      #5C6B57;
  --color-accent-deep: #48543F;
  --color-accent-soft: #EBEDE7;
  --color-warm:        #B0875A;
  --color-line:        #E3DFD5;
  --font-sans:  'Noto Sans JP', system-ui, sans-serif;
  --font-serif: 'Noto Serif JP', 'Yu Mincho', 'Hiragino Mincho ProN', Georgia, serif;
}
```

- 기존 `--color-accent: #7c3aed` 및 `--color-accent-soft: #ede9fe` 제거.
- `a { color: var(--color-accent); }` 유지 (변수로 자동 반영됨).

## STEP 3. 폰트 적용

### 3-1. Google Fonts 로드 (BaseLayout.astro `<head>`)
```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Noto+Serif+JP:wght@400;600;700&family=Noto+Sans+JP:wght@400;500;700&display=swap" rel="stylesheet">
```

### 3-2. 제목에 serif 적용
- Hero 제목(`.hero-title`), 성구(`.hero-verse`), 태그라인 → `font-family: var(--font-serif)`
- 카드 제목(`.link-card-label`) → `var(--font-serif)`
- 섹션 제목 전부 → `var(--font-serif)`
- 본문·버튼·UI → sans 유지

## STEP 6. 디테일 보정

- 모든 버튼 `border-radius: 4px` (기존 8px → 차분하게)
- 카드 아이콘: 이모지 유지 가능하나, 가능하면 **얇은 라인 아이콘**(SVG) 권장 (통일 시)
- `box-shadow`는 약하게: `0 4px 12px rgba(0,0,0,.05)`
- hover transform 유지 (`translateY(-1~2px)`)
